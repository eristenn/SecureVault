# SecureVault

SecureVault is a Python-based password vault project designed for learning cybersecurity, encryption, and secure software development concepts.

This project is being developed as a personal cybersecurity portfolio project and educational tool.

---

## Features

### Current Features
- Master password setup
- SHA-256 password hashing
- Login verification
- Command-line interface (CLI)

### Planned Features
- Encrypted credential storage
- Password generator
- SQLite database integration
- Fernet encryption
- Secure password viewing
- Password strength checker
- Auto-lock functionality
- GUI version

---

## Technologies Used

- Python
- hashlib
- cryptography
- SQLite3
